# sao-generator

A generator for scaffolding out a SAO generator.

## Features

- Bare minimal generator
- Unit test with [AVA](https://ava.li)

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao generator sao-name
```

### From repo

```bash
sao saojs/sao-generator sao-name
```
