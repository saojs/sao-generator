## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao <%= name.replace('sao-', '') %> my-project
```

### From git

```bash
sao <%= username %>/<%= name %> my-project
```
